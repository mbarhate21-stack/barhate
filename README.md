# User Management API Automation Testing Project

## Project Overview

This project demonstrates real-world API Testing using Postman, Newman, and JavaScript validations.

The project includes:
- CRUD API Testing
- Response Validation
- Schema Validation
- Negative Testing
- Authorization Testing
- Automated Collection Execution
- HTML Report Generation

---

# Tools & Technologies Used

- Postman
- Newman
- JavaScript
- Git & GitHub
- ReqRes API
- JSON Schema Validation
