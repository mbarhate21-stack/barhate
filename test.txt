GitHub is a web-based platform built on Git, a "version control system" that tracks changes to files. While Git runs on your local computer, GitHub serves as a cloud-based host for those projects, allowing you to store code online and collaborate with others.Core ConceptsRepository (Repo): A project's digital folder that contains all files and the complete history of every change made to them.Commit: A "snapshot" of your files at a specific point in time. It includes a message describing what was changed.Branch: A parallel version of a repository. You can create a branch to try new ideas or fix bugs without affecting the "main" project until you're ready.Pull Request (PR): A way to propose changes.
